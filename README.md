# SK海力士跨市场看板

## 上传 GitHub Pages

1. 新建一个 Public 仓库，例如 `skhynix-dashboard`
2. 解压本 ZIP
3. 将以下文件全部上传到仓库根目录：
   - index.html
   - manifest.webmanifest
   - sw.js
   - icon.svg
   - README.md
4. 进入 Settings → Pages
5. Source 选择 `Deploy from a branch`
6. Branch 选择 `main`，Folder 选择 `/(root)`
7. 保存后访问 GitHub 给出的 Pages 地址

## iPhone 添加到主屏幕

1. 使用 Safari 打开 GitHub Pages 地址
2. 点击分享
3. 选择“添加到主屏幕”

## 功能

- 韩国正股、香港07709、美国SKHY三地行情卡片
- 盘中、盘后、综合涨跌
- ADR与07709折溢价
- 当前相对最便宜市场高亮
- 折溢价阈值提醒
- 韩国盘后价推算香港与美国理论价格
- 任意输入一个市场目标价格，换算其余市场理论价格
- 自动保存输入、深色模式、复制行情摘要
