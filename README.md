# SK海力士交易看板（最新版静态版）

## 覆盖 GitHub
1. 解压 ZIP。
2. 打开原 GitHub 仓库。
3. 选择“添加文件 / Upload files”。
4. 上传压缩包内全部文件。
5. 同名文件会显示为修改，提交更改即可。
6. GitHub Pages 原网址不会改变。
